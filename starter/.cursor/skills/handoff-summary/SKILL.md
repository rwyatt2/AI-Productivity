# Handoff Summary

Use this when switching context (e.g. SPEC → IMPLEMENT) or handing off to
another person or session.

## Format: max 8 bullets

* What was decided or done
* What is in scope (and out of scope)
* Open questions or blockers
* Key files or artifacts
* Next recommended step
* _(Optional)_ Assumptions or constraints
* _(Optional)_ Security / exposure / data-sensitivity notes
* _(Optional)_ Where to look for more detail

Keep each bullet one line. No code. Fill only the bullets that apply.
