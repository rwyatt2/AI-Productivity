# Kit (canonical source)

This folder is the **kit-only** bundle. It contains:

* `.cursor/` — rules and prompts
* `docs/ai/` — config and source of truth
* `.github/` — PR template and Copilot instructions
* `cursor-ai-kit.config.json` — kit config

**Use this when:** You already have a repo. Copy everything inside `kit/` into your repo root, then fill placeholders in `docs/ai/ai-config.md`.

**Use the starter when:** You want a new repo with the kit already at the root. Copy the `starter/` folder instead.
